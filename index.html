AI Medicine Recommendation System

This project is based on Machine Learning. The model takes symptoms from the user and predicts the possible disease. It also provides suggestions for diet, exercise, training, description, medications, precautions and workouts. The model is trained on 136 symptoms using a Support Vector Classifier and achieves 99% accuracy. The model is deployed on a website built using Flask

Traffic Sign and Lane Detection for Autonomous Vehicles

Developed a prototype self-driving system using OpenCV and AI techniques. The model captures road video, detects lane edges using Canny edge detection, and refines them with segmentation and Hough Transform to highlight driving lanes. It uses a Spatial Convolutional Neural Network (SCNN) to detect and identify objects like vehicles and traffic lights. The system processes video input through a single-run script (run.bat) and simulates real-time driving decisions

Project link
IPL Prediction

Built a machine learning model to predict the winning team in IPL matches based on key match factors such as toss result, teams, city, runs left, balls left, wickets, and run rates. Used a Logistic Regression algorithm trained on real match data. Achieved a prediction accuracy of 96%, providing near real-time match outcome predictions. The project helps visualize the impact of game dynamics on win probability
